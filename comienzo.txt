# PROYECTO MULTIMARKET

Actúa como un Arquitecto de Software Senior especializado en Java, Spring Boot 3, PostgreSQL, Hibernate, JPA, Microservicios, Arquitectura DDD y sistemas Marketplace.

Tu tarea es diseñar y generar el modelo de dominio completo para el proyecto llamado MultiMarket.

## CONTEXTO DEL NEGOCIO

MultiMarket es una plataforma Marketplace Multi-Vendedor.

No es un e-commerce tradicional.

MultiMarket permite que múltiples vendedores independientes creen sus propias tiendas dentro de una misma plataforma, publiquen productos, administren inventario, reciban pedidos, interactúen con compradores mediante chat en tiempo real y procesen pagos.

El objetivo es conectar productores regionales y pequeños comerciantes con compradores a través de una única plataforma digital.

Conceptualmente

Marketplace
├── Tienda Vendedor A
├── Tienda Vendedor B
├── Tienda Vendedor C
└── Tienda Vendedor D

Cada vendedor administra

 Productos
 Inventario
 Pedidos
 Imágenes
 Conversaciones
 Importaciones XML
 Exportaciones JSONXML
